# NetIntent ONOS Dataset

This directory contains an unmodified snapshot of the NetIntent ONOS
Intent-to-Flow dataset used as an upstream research source.

## Source

- Repository: <https://github.com/Muhammadkamrul/NetIntent>
- Source path: `GitHub NetIntent/ONOS/Intent2Flow-ONOS.csv`
- Pinned commit: `5d5a2377673893dc8a6585346e64272da43a54fc`
- Local filename: `Intent2Flow-ONOS.csv`
- SHA-256: `d884b12355dcd4e203edec9b748023f24e334d61a9ff8129bdc1a593f29dcdfd`
- License: MIT; see `LICENSE.md` in this directory

The CSV is kept separate from this project's added and annotated cases. Its
ONOS flow JSON labels are controller-specific and are not presented as this
project's controller-neutral Intent IR ground truth.
